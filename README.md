# Crypto

A modern cryptocurrency tracking web app built with **Next.js** and **TypeScript**. View live market data, trending coins, and detailed information through a clean and responsive interface.

## Features

- Live cryptocurrency prices
- Trending coins
- Coin details and market statistics
- Responsive UI
- Fast and modern interface

## Tech Stack

| Technology | Usage |
|------------|-------|
| Next.js | Framework |
| TypeScript | Language |
| Tailwind CSS | Styling |
| API | Crypto market data |

## Installation

```bash
git clone https://github.com/rutuja2005byte/Crypto.git
cd Crypto
npm install
npm run dev
```
